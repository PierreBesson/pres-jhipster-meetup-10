Monolith vs Microservices

  • Monolith:
    • One application (Back-end + Front-end)
  • Microservice architecture:
    • Registry
    • API Gateway (proxy + front-end)
    • Microservices (back-end only)
    • (Optional) OAuth2 server UAA,Keycloak...

JHipster Microservice Architecture

How do I set up my microservice architecture with JHipster?

1) Generate your gateway and microservices apps

2) Start the JHipster registry

docker-compose -f src/main/docker/jhipster-registry.yml

3) Start your microservices (./mvnw or ./gradlew)

4) Enjoy
